主な使用技術

Angular

Webアプリケーションのフロントエンド開発にはAngularを使用しています。 大部分の工程が自動化されており、ReactやVueなど他のフレームワークを使ってきた方でも、フロントエンド開発の理解があれば対応が可能です。

TypeScript

Angularを使用するため、フロントエンド開発にはTypeScriptを使用しています。 簡易的なサーバーサイドプログラムもTypeScriptを使用することがあります。

Rust

機械学習(ベイズ統計学)の研究開発にはRustを使用しています。

Golang

Cosmos SDKを使ったブロックチェーン開発にはGolangを使用しています。

Git,GitHub

今や常識のソースコード管理としてGit,GitHubを使用しています。 CI/CDワークフロー自動化としてGitHub Actionsを使用しています。

Docker

サーバーサイドアプリケーションの管理にDocker, Docker Composeを多用しています。
